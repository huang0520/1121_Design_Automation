# Project 1 Using DP to solve TSP
## How to execute
```
# /project1

./bin/$executable $input_file_path $output_file_path
```

For example:
```
# /project1

./bin/project1 ./input/public_input1.txt ./output/public_output1.txt
```

## How to compile
1. 通過 Makefile
   ```
   # /project1

   make -C ./src/ 
  
   # or
   
   (cd ./src/ && make)
   ```
1. 通過 CMake + VSCode
